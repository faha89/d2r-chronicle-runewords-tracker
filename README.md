# D2R Chronicle Runeword Tracker

A lightweight, single-page tracker for the **Chronicle: Make All Runewords**
achievement in Diablo II: Resurrected (Reign of the Warlock DLC).

## Features

- Check off runewords as you craft them
- Live progress bar (X / 98 completed)
- Remaining runes panel — updates in real time as you check off runewords
- Filter by: All / Remaining / Completed / Ladder-only / Reign of the Warlock
- Search by runeword name
- Progress saved automatically in your browser (localStorage)

## Usage

Open directly in a browser — no server or install needed:

**https://faha89.github.io/d2r-chronicle-runewords-tracker**

## Runeword Sources

- Base game + ladder runewords: [d2r.guide](https://d2r.guide)
- Reign of the Warlock runewords: [Icy Veins](https://www.icy-veins.com/d2)
